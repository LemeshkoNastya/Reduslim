### Установка проекта
```
# Клонировать репозиторий
git clone https://github.com/LemeshkoNastya/Reduslim

# Перейти в репозиторий
cd Reduslim

# Установить зависимости
npm install

# Сборка проекта в режиме разработки
npm run dev

# Сборка проекта в режиме продакшн
npm run build
```
